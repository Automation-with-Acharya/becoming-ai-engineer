# Project ₹50L Contract (Version 1.0)

**Effective Date:** 11 July 2026

**Contract Duration:** 12 Weeks (Phase 1)

**Status:** 🔒 Frozen

## Purpose

This contract freezes the execution strategy for the first 12 weeks of
Project ₹50L. The roadmap may receive execution improvements, but the
overall direction, milestones, technologies, and sequence remain
unchanged.

## What "Frozen" Means

-   No switching languages because of trends.
-   No abandoning projects halfway.
-   No restarting with a new roadmap.
-   No changing technologies every week.
-   No redesigning the curriculum.

We execute the plan.

## What CAN Change

Execution quality may improve while keeping the same milestone.
Example: - Learn FastAPI → Convert Student Management into a REST API. -
Learn Docker → Dockerize the Student Management System.

## Permanent Rules

1.  Every topic ends with code.
2.  Every day produces GitHub commits.
3.  Every learning session produces Engineering Handbook notes.
4.  Every week produces one tangible improvement.
5.  Every month produces one portfolio-worthy project or major upgrade.
6.  Every technology connects to the current project.
7.  Official documentation first; videos second.
8.  Prefer quality over quantity.
9.  Optimize for interviews and production engineering, not
    certificates.
10. Learn to build systems, not just technologies.

## North Star Architecture

React Frontend ↓ FastAPI Backend ↓ Business Logic ↓ Repository Layer ↓
PostgreSQL ↓ AI Layer (RAG / LLM) ↓ Docker + Cloud Deployment

## Working Agreement

Daily: - Learning - Practice - Build - GitHub Commit - Daily Log

Weekly: - Review - Refactoring - Documentation

Monthly: - Portfolio Review - Resume Improvements - Interview Readiness

Quarterly: - Strategic review only if truly required.

## Mentor Commitment

Every recommendation will be evaluated against one question:

"Will this increase the probability of achieving a ₹25--30 LPA offer in
the near term and keep the path toward ₹50+ LPA?"

If yes, we do it. If no, we don't spend time on it.

## Final Commitment

From 11 July 2026, the Phase 1 (12-week) roadmap is officially frozen.

We stop redesigning. We execute.
