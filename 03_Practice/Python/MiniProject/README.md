# Student Management CLI

## Objective

This project was created to build a simple command-line application for managing student records. The main goal is to practice core Python concepts such as functions, dictionaries, loops, and user input handling while creating a small real-world utility.

## Features

- Add a new student with an automatically generated unique ID
- View all stored students with their IDs and names
- Search for a student by ID or name
- Delete a student by ID or name
- Use a simple menu-driven interface for easy interaction
- Handle basic input validation and user-friendly messages

## Concepts Used

- Functions for organizing program logic
- Dictionaries to store student data as key-value pairs
- Lists and list comprehensions for filtering and searching
- Loops to keep the program running until the user exits
- Conditional statements for menu choices and validation
- Basic exception handling with try/except

## Learnings

- Learned how to build a small CLI project from scratch using Python
- Understood how to use dictionaries effectively for storing structured data
- Practiced creating reusable functions for common operations
- Improved my ability to handle user input and respond to invalid choices
- Gained confidence in writing logic for searching and deleting records

## Future Improvements

- Save student data to a file so it persists after program exit
- Add an option to update or edit student information
- Improve search to support partial name matching
- Add sorting options for viewing students
- Enhance the interface with better validation and error messages
