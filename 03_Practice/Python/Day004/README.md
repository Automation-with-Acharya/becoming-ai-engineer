# Student Management CLI v3

This folder contains a class-based version of the student management project.

## Files

- student.py: defines the Student class
- student_manager.py: defines the StudentManager class and the CLI menu
- students.txt: stores student data on disk

## How to run

From this folder, run:

```bash
python student_manager.py
```

## Why classes help

Classes help organize related data and behavior in one place.
A Student object stores the student details, while StudentManager handles the operations such as add, search, delete, view, load, and save.
