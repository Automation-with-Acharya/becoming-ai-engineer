Backend

Python

- Flask
- FastAPI
- Django

Use Django when:

- Admin Panel required

- Lightweight application
- Faster development
- More flexibility

# Lessons Learned

1. Real interviews focus heavily on architecture and problem-solving.
2. Explain "why" a technology is chosen, not just "what" it is.
3. Use measurable achievements while introducing yourself.
4. Be honest about technologies you are still learning.
5. Asking thoughtful questions leaves a positive impression.
6. Every interview reveals the next set of skills to master.
