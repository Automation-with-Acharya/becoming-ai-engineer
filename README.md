# Career Transformation Roadmap

## Master Goal

- Achieve a salary package of 50+ LPA in 1 year — July 2027.

## Learning Roadmap

### Phase 1 — Build Foundations (Weeks 1–12)

- Goal
  - Become a top-tier backend engineer.
- Deliverables
  - Python mastery
  - FastAPI
  - PostgreSQL
  - Docker
  - Redis
  - Authentication
  - Testing
  - CI/CD basics

### Phase 2 — Become an AI Engineer (Weeks 13–24)

- Deliverables
  - Retrieval-Augmented Generation (RAG)
  - AI agents
  - Model Context Protocol (MCP)
  - LangGraph
  - Vector databases
  - LLM deployment
  - Multi-agent systems

### Phase 3 — Senior Engineer Skills (Weeks 25–38)

- Deliverables
  - System design
  - Kubernetes
  - AWS
  - Terraform
  - Monitoring
  - Scalability

### Phase 4 — Market Positioning (Weeks 39–52)

- Deliverables
  - GitHub profile
  - LinkedIn profile
  - Open source contributions
  - Resume
  - Interview preparation
  - Salary negotiation
  - Product company applications

## Projects

### 1. Enterprise AI Assistant

- Goal
  - Build a business-grade AI assistant for enterprise workflows.
- Features
  - Chat interface
  - RAG over company documents
  - Memory
  - SQL querying
  - Email drafting
  - Role-based access
  - Admin dashboard
  - Docker packaging
  - CI/CD pipeline
  - Cloud deployment

### 2. AcharyaVerse 2.0

- Goal
  - Create a recruiter-focused personal portfolio experience.
- Features
  - 3D portfolio
  - AI assistant trained on your resume
  - Voice interaction
  - Live GitHub integration
  - Dynamic project showcase
  - Blog
  - Recruiter mode
  - Architecture visualization
- Note
  - This should become something recruiters remember after one visit.

### 3. AI Automation Platform

- Goal
  - Build the next evolution of UiPath-style automation with AI orchestration.
- Concept
  - Users describe workflows in plain English.
  - The platform orchestrates APIs, browser automation, document understanding, approvals, and agents.

### 4. AI DevOps Copilot

- Goal
  - Build an AI assistant that supports DevOps workflows.
- Capabilities
  - Writes code
  - Reviews pull requests
  - Writes tests
  - Deploys applications
  - Opens pull requests
  - Reads logs
  - Detects failures
  - Explains root causes
  - Suggests fixes
  - Creates tickets
  - Generates summaries

### 5. SaaS Product

- Goal
  - Identify a real pain point and build a product people actually use.
- Value
  - Even modest traction teaches product thinking, operations, and customer feedback.

## Tech Stack

- Backend: Python, FastAPI, PostgreSQL
- Infrastructure: Docker, Kubernetes, AWS, Terraform
- AI: RAG, vector databases, LLM deployment, AI agents
- DevOps: CI/CD, monitoring, cloud deployment
- Product & positioning: GitHub, LinkedIn, open source, resume, interviews
